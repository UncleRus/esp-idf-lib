## Connecting

| `TSL2591` | `ESP8266` | `ESP32` | `NodeMCU` |
|-----------|-----------|---------|-----------|
| `SCL`     | `GPIO5`   | `GPIO19`| `D1`      |
| `SDA`     | `GPIO4`   | `GPIO18`| `D2`      |
| `INT`     | `GPIO2`   | `GPIO2` | `D4`      |
